## 6-9, 6-10 详情弹层页 - 实现弹出层

> sticky layout: A,B,C,D,E,F

```html
<div class="detail">
  <!-- A: A零件 clearfix -->
  <div class="detail-wrapper clearfix"></div>
  <!-- B: 切分为 A B 两个部分 -->
  <div class="detail-close"></div>
</div>
```

```css
.detail-wrapper {
  /* 内容长度会改变 */
  width: 100%;
  /* C: 内容长度会改变 */
  min-height: 100%;
  .detail-main {
    /* D: A part space for B part */
    padding-bottom: 64px;
  }
  .detail-close {
    position: relative;
    /* E: B零件边距嵌入A零件的内边距 */
    margin: -64px auto 0;
    /* F: B零件 clearfix */
    clear: both;
  }
}
```
